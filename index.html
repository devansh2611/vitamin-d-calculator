About Vitamin D

October 1, 2017

Vitamin D is known to be important for the human body. There are many benefits of not having a vitamin D deficit, which is supported by many research documents in the domain. However, vitamin D3 is supposed to be more efficient in humans than vitamin D2.
Thus, this site is providing a calculator for the right vitamin D3 doses, when a vitamin D3 supplement is intended.

Vitamin D3 Calculator

October 2, 2017

The goal of the calculator is to help identify the right vitamin D3 and vitamin K2 doses. This is done based on providing input data related to the current bodyweight and the current vitamin D3 level.
The calculator is set to use a guide value of 50 IE/kg
